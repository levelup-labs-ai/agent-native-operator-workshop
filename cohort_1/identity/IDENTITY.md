# Identity

This file is how your Claw appears to you. OpenClaw reads it at the
workspace root to set the bot's name, mascot, and presentation.

If any field still has placeholder text (`<...>`), the install hasn't
been customized yet. The customize-identity step in Coworking 1
fills these in.
